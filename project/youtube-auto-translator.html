YouTube
Auto-Translator

클릭 한 번으로 26개 언어 자막을 자동 번역·등록

YouTube 스튜디오의 자막을 Google Gemini AI가 자동으로 번역해 각 언어의 자막 트랙에 바로 업로드합니다. 반복 작업 없이 글로벌 시청자에게 다가가세요.

Features

주요 기능

번거로운 다국어 자막 작업을 한 번에 처리합니다.

🌍

26개 언어 동시 번역

영어, 일본어, 중국어(간체·번체), 스페인어, 프랑스어 등 주요 언어를 한 번에 처리합니다.

✨

Gemini AI 번역 품질

Google의 최신 AI 모델(Gemini 2.5 Flash)을 활용해 자연스럽고 문맥에 맞는 자막을 생성합니다.

⚡

YouTube Studio 직접 등록

번역 완료 즉시 YouTube 스튜디오의 자막 트랙에 자동으로 업로드됩니다. 별도 파일 작업이 필요 없습니다.

🔒

API 키 완전 로컬 저장

Gemini API 키는 사용자의 브라우저에만 저장됩니다. 어떤 외부 서버로도 전송되지 않습니다.

📋

번역 이력 관리

사이드바에서 이전 번역 결과와 진행 상황을 언제든 확인할 수 있습니다.

🆓

Gemini 무료 티어 지원

Google AI Studio의 무료 API 키로 바로 사용할 수 있습니다. 소규모 채널도 부담 없이 시작하세요.

How to Use

사용 방법

설치부터 첫 번역까지 5분이면 충분합니다.

1

Gemini API 키 발급

Google AI Studio에서 무료로 API 키를 발급받습니다.

2

Chrome 웹 스토어에서 설치

Chrome 웹 스토어에서 "YouTube Auto-Translator"를 검색해 설치합니다.

3

API 키 입력

확장 프로그램 설정 페이지에서 발급받은 Gemini API 키를 입력하고 저장합니다.

4

YouTube Studio에서 번역 시작

YouTube 스튜디오의 영상 수정 페이지로 이동하면 사이드바가 자동으로 열립니다. 번역할 언어를 선택하고 [번역 시작]을 클릭합니다.

5

자막 자동 등록 완료

번역이 완료되면 선택한 언어의 자막이 YouTube 스튜디오에 자동으로 등록됩니다.

📌 참고
번역하려는 영상에 원본 자막(원어 자막 트랙)이 먼저 등록되어 있어야 합니다. Gemini API는 무료 티어를 제공하지만, 사용량에 따라 요금이 발생할 수 있습니다. 자세한 요금은 Google AI 요금 안내를 참고하세요.

지금 바로 글로벌 채널로

Gemini API 키 하나로 26개 언어 자막을 자동화하세요.
