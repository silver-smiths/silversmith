JSBin.kr

로그인 없이 바로 — HTML · CSS · JS를 쓰고 공유하세요

jsbin.com 서비스 종료 이후 많은 개발자들이 찾던 바로 그것. 회원가입도, 설치도 없이 브라우저에서 바로 코드를 작성하고 URL 한 줄로 스냅샷을 공유할 수 있는 무료 온라인 코드 에디터입니다.

Features

주요 기능

복잡한 설정 없이 코딩에만 집중할 수 있습니다.

🆓

로그인 없이 바로 시작

회원가입, 이메일 인증, 설치 — 아무것도 필요 없습니다. 브라우저를 열고 바로 코딩을 시작하세요.

🖥️

3패널 동시 편집

HTML · CSS · JS를 나란히 놓고 편집합니다. 각 패널은 독립적으로 크기를 조절할 수 있습니다.

⚡

실시간 미리보기

코드를 실행하면 샌드박스 iframe에서 결과를 즉시 확인합니다. 수정할 때마다 빠르게 피드백을 받으세요.

🔗

URL 한 줄로 공유

공유 버튼 하나로 코드 스냅샷 URL을 생성합니다. 링크를 붙여넣으면 누구나 같은 코드를 볼 수 있습니다.

✨

CodeMirror 에디터

신택스 하이라이팅, 자동 들여쓰기, 괄호 자동 완성 등 개발자가 기대하는 편의 기능을 갖췄습니다.

🔒

안전한 코드 실행

사용자 코드는 샌드박스 iframe 안에서만 실행됩니다. 상위 페이지와 완전히 격리되어 안전합니다.

How to Use

사용 방법

접속부터 공유까지 30초면 충분합니다.

1

jsbin.kr 접속

브라우저에서 jsbin.kr로 접속합니다. 설치나 로그인 없이 바로 에디터가 열립니다.

2

코드 작성

HTML · CSS · JS 패널에 원하는 코드를 자유롭게 작성합니다. CodeMirror 에디터가 신택스 하이라이팅과 자동 완성을 지원합니다.

3

Run으로 실행 · 미리보기 확인

Run 버튼을 누르면 오른쪽 미리보기 패널에 결과가 즉시 표시됩니다. 코드를 수정하고 다시 실행해 반복적으로 확인하세요.

4

공유 URL 생성

공유 버튼을 클릭하면 현재 코드 스냅샷의 고유 URL이 생성됩니다. 링크를 복사해 동료나 커뮤니티에 바로 공유하세요.

📌 참고
JSBin.kr은 광고 지원 모델로 운영됩니다. Run 실행 시 보상형 광고가 표시될 수 있으며, 광고 시청 후 미리보기가 열립니다. 덕분에 모든 핵심 기능을 완전 무료로 제공합니다.
